# Employee Task Tracker

A responsive Employee Task Tracking system built with React and Vite.

## Features
- Add tasks and assign to employees
- Filter tasks by status (Pending / In Progress / Completed)
- Search tasks by title
- Sort tasks (newest / oldest)
- Update task status, delete tasks with Undo
- Dashboard with completion metrics and radial progress
- Local persistence via localStorage
- Polished UI with animations, badges, and accessibility improvements

## Tech Stack
- React + Vite
- Bootstrap (react-bootstrap)
- LocalStorage for persistence

## Run locally
1. `npm install`  
2. `npm run dev`  
3. Open `http://localhost:5173`

## Build & Deploy
- `npm run build` then serve `dist/` or deploy to Vercel / Netlify / GitHub Pages.
